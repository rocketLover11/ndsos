ndsos
=====
By: rocketLover11

### Description
ndsos is an operating system for the Nintendo DS. ndsos is also my first DS project. This is a hobby project so don't judge, I thought it would be fun to make.

### Build Instuctions
This guide assumes you have CMake, devkitPro, and devkitARM installed and DEVKITPRO and DEVKITARM environment variables set.

```sh
git clone https://github.com/rocketLover11/ndsos.git
cd ndsos
cmake -B build
cmake --build build
```
